
# SignalWire Coding Challenge
Coding asssignment for SignalWire related to `Websockets`, `TypeScript`, and `NPM Workspaces`.



## Getting Started

**1:** Install dependencies
```
npm i
```
**2:** Build the project
```
npm run build
```

## Running Workspaces

**Node SDK**

```
npm start -w @signalwire/node-sdk
```

**Browser SDK**

```
npm start -w @signalwire/browser-sdk
```
